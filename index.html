Tetris

My first graphical game, and my first javascript. Started with basic game logic from a youtube how to, have added many new features including game menus, game states, high score saving, game stats, sounds, mobile device touch controls, and more.

Some possible features include improved stats tracking, traditional scoring model, options for game start, or improved menus.
